# Plan 01-03: Create Proxy Setup Documentation

## Objective

Document the correct way to set up proxies for Reddit account farming, based on research and audit findings.

## Context

Freelancers need clear documentation on:
1. How to configure DataImpulse/Decodo for Reddit
2. Why sticky sessions matter
3. How to match timezone with proxy geo
4. What NOT to do (common mistakes)

## Tasks

<task id="1">
Create `dolphin/docs/PROXY_SETUP.md`:

## DataImpulse Setup
- Where to get credentials (dashboard)
- Proxy format: `http://user__geo_params:pass@gw.dataimpulse.com:PORT`
- Port selection:
  - **823** = Rotating (DON'T USE for Reddit)
  - **10000+** = Sticky sessions (USE THIS)
- Geo targeting: `__cr.us;state.california;city.losangeles`

## Decodo Setup
- Proxy format: `http://user-sessionduration-60:pass@gate.decodo.com:7000`
- Session duration: minimum 60 minutes recommended
- Why: Reddit flags frequent IP changes

## Common Mistakes
- Using rotating proxies (looks suspicious)
- Sharing proxies between accounts
- Timezone mismatch
</task>

<task id="2">
Create `dolphin/docs/DOLPHIN_CONFIG.md`:

## Creating a New Profile
1. Click "Create Profile"
2. Set timezone to match proxy geo (e.g., CST for Illinois)
3. Add proxy with sticky session
4. Click "Check Proxy" - verify location matches
5. Generate fingerprint
6. Save and launch

## Cookie Settings
- Keep cookies between sessions (don't clear)
- Export cookies before risky operations
- Import cookies to recover session

## What to Verify
- [ ] Timezone matches proxy location
- [ ] Sticky session (not rotating)
- [ ] Proxy test passes
- [ ] Unique fingerprint generated
</task>
